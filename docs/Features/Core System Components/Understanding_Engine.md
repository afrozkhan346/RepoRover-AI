# Understanding Engine — Feature Coverage

**Features:**

- Entry point detection
- Core function identification
- Project type inference

**Working:**

- Entry point detection (main file, etc.) — working
- Core function identification (graph-based ranking) — working
- Project type inference (by file extensions and structure) — working

**Not working / Not found:**

- No support for custom entry point rules
- No deep semantic project type classification (beyond file extensions)

**Summary:**
All listed features are implemented and working. Custom rules and advanced project type inference are not covered.
